PAO system

Person Action Object system

This is a system that can be used to encode the order of cards or numbers to make it easier to memorise them. The way it works is that every card has an associated person, action and object. These are usually related to each other to aid recalling them. For example the 3 of hearts could be Michael Jordan shooting a basketball. Here are some more examples:

Card Person Action Object
Queen Hearts Serena Williams serving tennis ball
Ace Diamonds Michelangelo chiseling the David
3 Diamonds Michael Jackson moonwalking white glove
2 Spades Neil Armstrong planting USA flag
8 Clubs Frodo putting his finger in the ring

As you can see actions and objects clearly fit with the person, and the person can be a fictional character or a real person. This association needs to be done for all 52 cards in a deck or all numbers from 00 to 99. The aim here is to be as creative as possible and get a wide range of person, actions and objects to minimize overlap which could become confusing. There are some tricks to making this process easier:

  1. Choose categories. For the cards you can decide to make each suit associated with a certain category. For example for my system I have chosen Hearts to be athletes, Diamonds to be artists, Spades to be historical figures and Clubs to be fictional characters. These categories can be much more specific however, such as only characters from a tv show or a book, or only politicians. Do realise that having variety in the persons makes it easier to choose fitting actions and objects (Try finding 12 actions that a politician could do?)
  2. Use the Major system to associate card numbers with person names. This will associate each number with certain sounds which you can use to find names that start with that sound, for example 3 diamonds -> Michael Jackson. The major system can be a good way to help your memory along when trying to remember the name of the person a card is associated with.
  3. Test out your action and object combinations. The actions and objects will need to be combined in all possible combinations, so making sure that you can somehow visualize them going together is a good check. Don't let this stop you from getting creative however, it is often the weird combinations that stick the best in your memory.
  4. Get explicit and rude. Using sexual or violent actions might not be nice but they tend to stick better than others. The same goes for polarizing figures, or people you have a strong emotional reaction to. (Try including Hitler, Trump or your secret crush).
  5. Don't be afraid to break the rules. If you have a really good association between a card and a person but it doesn't fit in the right category, you can still use it. We are going for strong associations here so if you find one that you like, keep it!

Once every card has a PAO associated with it, we can combine this system with a memory palace to memorize the order of a deck of cards. sort the cards into sets of three, then for each set of three cards take the person of the first, the action of the second and the object of the third card. Try to form one coherent image out of these three cards and place this image at the first location in your memory palace. Play around with the image a bit or maybe even try talking to the person to really make it stick. Then move on to the next set of three cards and place them at the second location. Keep doing this until you are through the deck. For a standard deck of cards you will need 18 locations, where the last location will have just one card. Depending on how you wish to remember the card order you coul omit the last location as this will simply be the last card left (this is for if you want to put a second deck into the same order, if you want to recite the order of the deck you better memorize that last card to avoid an anticlimactic ending).

The PAO system for a deck of cards can also be repurposed for numbers by using the numbered cards (and aces) for 01-40 and then finding a way to sort the name cards into the next 12 numbers.
